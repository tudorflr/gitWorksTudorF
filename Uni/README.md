# University Projects

This folder contains academic projects and laboratory work from various Computer Science courses.  
The projects focus on core concepts such as algorithms, systems, low-level programming, and software design.

## Structure

### Algorithms and Data Structures
- DataStructuresAlgorithms  

Focus:
- graph algorithms (BFS, DFS, Dijkstra, A*, Prim, Kruskal)
- tree operations and traversals (preorder, leaf counting)
- hashing and data structures (hash tables, custom structures)
- problem solving using divide and conquer and dynamic programming (knapsack, rod cutting, maximum subarray)
- working with structured input data and algorithmic processing

---

### Fundamental Algorithms
- FundamentalAlgorithms  

Focus:
- implementation and comparison of sorting algorithms (bubble, insertion, quicksort, heapsort)
- tree structures and analysis (BST, multiway trees)
- graph traversal and ordering (BFS, DFS, topological sorting)
- disjoint sets and minimum spanning tree algorithms (Kruskal)
- performance analysis and comparison of algorithmic approaches

---

### Operating Systems
- OperatingSystems  

Focus:
- Linux system programming in C
- low-level file handling, directory traversal, and custom file parsing
- process creation and hierarchical execution with fork and wait
- multithreading with pthreads and synchronization using semaphores
- IPC mechanisms including named pipes, shared memory, and mmap

---

### Computer Architecture
- ComputerArchitecture  

Focus:
- 16-bit MIPS CPU design in VHDL (ALU, register file, control unit)
- datapath design and control logic implementation
- instruction execution flow and memory interaction
- extension to a pipelined architecture for improved performance

---

### Java OOP Projects
- JavaOOP  

Focus:
- object-oriented design
- collections and data structures
- serialization and basic multithreading

---

### Computer Graphics
- ComputerGraphics  

Focus:
- graphics fundamentals using SDL and C++ (vectors, matrices, geometric processing)
- 2D and 3D transformations, including object manipulation and camera mapping
- rasterization techniques (Bresenham, line drawing, pixel mapping)
- clipping algorithms (Cohen-Sutherland, Cyrus-Beck)
- depth handling and rendering concepts (Z-buffer, barycentric coordinates)

---

### Assembly
- Assembly  

Focus:
- low-level programming
- understanding hardware-software interaction

---

## Notes

These projects were developed as part of university coursework and focus on learning core computer science concepts.
